### Brainfuck implementation in golang
#### About
[Wikipedia](https://en.wikipedia.org/wiki/Brainfuck) [esolang](https://esolangs.org/wiki/Brainfuck)
#### Build
```sh
go build bf.go
```

#### Usage
Input from output:
```sh
./bf -utf -file="path/to/file"
```

Input from stdin:
```
./bf
<here paste your bf program>
<Ctrl+D to end input>
```
